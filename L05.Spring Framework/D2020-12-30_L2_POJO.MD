
# POJO 란 무엇인가?

- 자바 언어와 필수적 API 외에는 규약에 종속되지 않는다
- 특정 환경에 종속되지 않는다.
- 스프링에서는 스프링에 특화된 인터페이스 구현을 요구 하지 않는다
- 스프링 자체에 의존성이 높은 클래스 확장을 거의 요구 하지 않는다

- 스프링은 엔터프라이즈 개발에서 요구하는 모든 기술을 POJO를 통해 제공한다
- 비침투적 프로그램이 가능하다.
